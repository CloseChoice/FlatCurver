# FlatCurver

## Prerequisites

-   node.js > v12.16
-   npm > v6.13

## Project Installation

```
npm install
```

## Project Run

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
